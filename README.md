# prosperomerge
prosperomerge is a GUI tool made using [LuaRT](https://github.com/samyeyo/LuaRT) for merging parts of PS5 app/game updates into one installable .pkg file 

# First launch is long
First bootup might take a little longer because the app fetches information about your updates from [PROSPEROPatches](https://prosperopatches.com/)

# I found a bug or i have a suggestions
Go to the issues tab and post your idea there

# Closed source?
Yep currently im not planning on making it open source

# How to use?
Simple. 
1. Go to https://prosperopatches.com/ and search for an update you want
  # <img width="1073" height="509" alt="image" src="https://github.com/user-attachments/assets/668ab864-b874-44d4-8f9f-446ebde3c49f" />
2. Click on the update version you want to download and make sure the firmware requirement fits your PS5
  # <img width="678" height="281" alt="image" src="https://github.com/user-attachments/assets/c3a2aff7-15f7-4e8c-a0de-53a0ef32b8f5" />
3. Download DP, SC, and all piece parts files
  # <img width="451" height="543" alt="image" src="https://github.com/user-attachments/assets/e2b78930-5637-4faa-9396-145229e8f156" />
4. Extract the downloaded .zip file from releases and put the ProsperoMerge folder somewhere safe for ex. your desktop
   # <img width="100" height="104" alt="image" src="https://github.com/user-attachments/assets/b64fdfbb-d7c4-463a-bb3e-8dcec9e7f0e2" />
5. Put all of your downloaded files into the extracted folder (ProsperoMerge/updates)
 # <img width="650" height="246" alt="image" src="https://github.com/user-attachments/assets/2f73329b-472f-4489-aa2f-abc9ef01c2b6" />
6. Open prosperomerge.exe and click on the update you want to merge
 # <img width="586" height="139" alt="image" src="https://github.com/user-attachments/assets/a0bfc8f2-d0d9-4f20-9575-ed8b5ed43114" />
 # <img width="600" height="207" alt="image" src="https://github.com/user-attachments/assets/57b09738-e1f7-4e90-aa21-77cbe8eac770" />
7. After the merging finishes (progressbar and cmd window disappears) your file will be placed at ProsperoMerge/output
 # <img width="628" height="159" alt="image" src="https://github.com/user-attachments/assets/5e5c7890-495b-4f2c-adcf-49dfaa839a15" />
8. Now put it on a USB storage drive of your choice and using Debug Settings on your PlayStation go to the Package Installer
 # <img width="1195" height="700" alt="image" src="https://github.com/user-attachments/assets/147c2118-8883-4eb1-9544-84ec0dee5152" />
 # <img width="1134" height="280" alt="image" src="https://github.com/user-attachments/assets/c96fb48e-5418-4280-9476-09299bec9add" />
9. After that select the .pkg file you put on your USB drive and wait until it installs
Enjoy!
